Limbajul HTML

HTML(HyperText Markup Language)este un limbaj de marcare utilizat pentru a crea pagini web.

Un exemplu de cod in html il gasiti in imaginea urmatoare care se afla la adresa https://upload.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png.
Imaginea are atributul height de valoare 147.

Tag-urilehtml cele mai utilizate in realizarea tabelelor sunt:

table defineste un table
tr un rand dintr-un tabel
td o celula dintr-un tabel

Observatie:Pentru a pune un contur unui tabel se poate (dar nu se recomanda) sa puna atributul border="1".
La validare va da eroarea:The border attribute on the table element is obsolente.

Formularele se pot crea cu ajutorul tag-ului form care contine tag-urile:input, textarea, label, select, option, button.
Pentru mai multe detalii consultati cursul sau tutorialul HTML Forms and Inputs de la adresa http://www.w3schools.com/html/html_forms.asp.

Pentru a crea liste se pot folosi tag-urile:

  1. ul-lista neordonata
  2. ol-lista ordonata
  3. li-item in list
  4. dl -lista cu descrieri
  5. dt-termen dintr-o lista cu descrieri
  6. dd-descrierea dintr-o lista cu descrieri

Pentru a insera un document HTML in altul se poate folosi tag-ul < iframe>.
